# UniVoice 🎙️ | Multilingual Real-Time Voice Collaboration & Translation Hub

UniVoice is an elite, real-time voice collaboration platform engineered to break language barriers in live sessions and eliminate the need for traditional hardware microphone setups. By turning any mobile device or browser into an intelligent, localized digital microphone, UniVoice allows a single coordinator to seamlessly manage hundreds of participants across multilingual environments.

## 🚀 Key Value Propositions & Innovations

* **Hardware Microphone Replacement:** Completely eliminates the logistical overhead of manual physical microphones in classrooms or meeting spaces. Every participant's smartphone or browser acts as a smart, digital microphone node.
* **Real-Time Multilingual Translation Engine:** Powered by advanced speech-to-text models, the system automatically intercepts and translates spoken Urdu, Hindi, English, or any code-switched mix into crisp, instantaneous English captions.
* **Intelligent Audio Overlap Prevention:** Features a smart audio-ducking pipeline that can read translated captions aloud while muting the original speaker's raw audio track to ensure clear, clash-free playback for international audiences.
* **Granular Moderator Control Panels:** Gives coordinators absolute governance over live interaction parameters, including text/vocal portal toggles, custom mic timers, hand-raise queues, live polls, and emoji streaming layers.
* **Universal Fluid Layout:** A fully responsive, cross-platform architecture optimized for ultra-low latency execution across both mobile viewports and desktop workstations.

## ⚙️ Advanced Technology Stack

* **Runtime & Architecture:** Node.js, Express Framework
* **Real-Time signaling & State Synchronization:** Socket.IO (WebSockets) for event-driven, bi-directional network updates.
* **High-Fidelity Audio Pipelines:** WebRTC (RTCPeerConnection API) with custom browser-level programmatic constraints (echo cancellation, noise suppression, and auto-gain control).
* **AI Processing Infrastructure:** 
  * **Groq Whisper API:** For high-speed, low-latency multilingual speech recognition and translation.
  * **Groq Llama Models:** For contextual Q&A assistance and semantic intelligence feeds.
